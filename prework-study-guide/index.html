HTML

  • HTML is used to create the structure of a webpage.
  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user
  • The image tag allows us to add images to the webpage
  • The heading tags range from h1 to h6

CSS

  • CSS is a stylesheet that has to be linked to the HTML file in order to see the formatting results we desire.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • the period before card indicates that it is a class selector. Any HTML element assigned to the class, card, will gain these property assignments from the CSS.
  • the box shadow property requires two offsets- x axis and y axis.the function of box-shadow is self explanatory. it is used to create the shadow effect in a table on a webpage.
  • We can either use hexadecimal code to select the color we want or use semantic references.

Git

  • Git version control is used to keep track of content changes that I am going to make in my project(s).
  • git pull origin main is used to get to the main branch
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it

JavaScript

  • JavaScript is an Object-Oriented Programming language.
  • It is a popular coding language used to create interactive multimedia websites.
  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script, therefore we must code accordingly, for example we must define our global variables at the top.
  • When we create a function, we are creating a specific task by defining a block of code, and we have to give it a unique name so that it doesn't clash with built-in functions.
  • However, just defining a function does not mean it will execute in our code automatically, we must call it in our code.
